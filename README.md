# Weather Checker

A simple Python CLI tool to check weather information using free APIs (no API key required).

## Features

- 🌡️ Current temperature and conditions
- 💧 Humidity and wind speed
- 📅 3-day forecast
- 🌍 Works with any city name
- 🔑 No API key needed

## Usage

### Quick Start

```bash
python weather.py <city-name>
```

### Examples

```bash
# Check weather in London
python weather.py London

# Check weather in New York
python weather.py "New York"

# Check weather in Tokyo
python weather.py Tokyo
```

## Data Sources

- [wttr.in](https://wttr.in) - Weather data service

## Requirements

- Python 3.7+
- Internet connection
